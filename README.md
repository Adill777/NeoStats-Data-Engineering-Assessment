# NeoStats-Data-Engineering-Assessment

## Overview

This project implements an end-to-end ETL pipeline for retail sales data processing and business insights generation.

## Deliverables

* Python ETL Pipeline (`retail_pipeline.py`)
* Cleaned Dataset (`cleaned_retail_data.xlsx`)
* Power BI Dashboard (`Mohammed_Adil_NeoStats_Dashboard.pbix`)
* Detailed Project Documentation (`Mohammed_Adil_NeoStats_Data_Engineering_Assessment.pdf`)

## Technologies Used

* Python
* Pandas
* OpenPyXL
* Power BI
* VS Code

## Key Features

* Data Cleaning and Validation
* Missing Value Handling
* Category and Product Standardization
* PII Masking
* Revenue Calculation
* KPI Generation
* Interactive Power BI Dashboard
